# AI Resume Analyzer

AI Resume Analyzer is an AI powered resume evaluation tool that analyzes resumes, calculates ATS scores, identifies skill gaps, and provides personalized recommendations for career improvement.

## Features

* Resume Parsing
* ATS Score Calculation
* Skill Gap Analysis
* Resume Recommendation Engine
* Job Description Matching
* Flask-Based Web Interface

## Tech Stack

* Python
* Flask
* Natural Language Processing (NLP)
* HTML/CSS
* Git & GitHub
